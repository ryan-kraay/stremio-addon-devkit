    enum Stremio::Addon::DevKit::ContentType

    Overview

    These are the possible content types supported by Stremio, you can prune or expand this to fit your needs See: https://github.com/Stremio/stremio-addon-sdk/blob/master/docs/api/responses/content.types.md

    Defined in:

    stremio-addon-devkit/catalog.cr

    Enum Members

    Movie = 0

    movie: movie has metadata like name, genre, description, director, actors, images, etc.

    Series = 1

    series: has all the metadata a movie has, plus an array of episodes

    Channel = 2

    channel: created to cover YouTube channels; has name, description and an array of uploaded videos

    TV = 3

    tv: has name, description, genre; streams for tv should be live (without duration)
